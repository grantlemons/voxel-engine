## About

I wanted to do something challenging without looking at existing engines.
I have read some papers describing relevant datastructures and algorithms, but the implementations are all my own.

I am using some libraries, but not for core elements of the project, such as the datastructures.
Probably the biggest I'm willing to include is wgpu.

## AI Use & Contributors

AI has not been used and will not be used for this project.
I am doing this for the fun of it, not for the end product.

For the same reason, I am not accepting outside contributions, though opening issues is fine.
